# Overview

Store is a Kotlin Multiplatform library for building network-resilient applications.

- **[Essentials](#essentials)**: Step-by-step tutorials to help you through initial setup and fundamental Store concepts. Intended for folks new to Store. The purpose of these docs is learning.
- **[Capabilities](#capabilities)**: How-to guides with directions how to complete a specific task using Store. Intended for folks who've used Store before and have a specific task to complete. The purpose of these docs is task completion.
- **[Reference](#reference)**: Technical references describing in detail how Store works. The purpose of these docs is information retrieval.
- **[Explanation](#explanation)**: More detailed background, discussion, and conceptual guides. Intended for Store contributors. The purpose of these docs is understanding Store internals.

## Supported Platforms
Store5 currently supports the following Kotlin platforms:
- Android
- JVM
- iOS (ARM64, X64)
- Linux (X64)
- JS (Browser, Node.js)
